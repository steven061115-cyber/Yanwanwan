# 二游活动小灵通

一个专为二次元手游玩家设计的 iOS 活动提醒 app。追踪游戏限时活动倒计时，防止错过奖励。

---

## 功能

### 米哈游游戏（原神 / 崩坏：星穹铁道）
- 自动从 API 拉取活动日历，无需手动录入
- 活动按"一周内结束 / 一周后待做 / 已完成"分组展示
- 前瞻直播兑换码自动抓取并推送通知
- 搜索 + 类型筛选

### 任意自定义游戏
- 用户输入游戏名 + B 站版本公告链接
- AI（Jina Reader + DeepSeek）自动提取活动名称和起止时间
- 每次新版本上线后，刷新按钮重新粘贴链接即可更新

### 活动攻略
- 每条活动可跳转攻略页，B 站相关视频
- AI 查询完成该活动大约所需时长


### iOS 系统通知提醒
- **每日提醒开关**：用户可在设置页独立开启 / 关闭每日提醒，关闭时仅取消每日提醒，6 小时紧急提醒不受影响
- 每日自定义时间通知：在用户设定时间（支持 24 小时内任意时刻）提醒即将过期的活动，次日起 7 天内预注册
- 最后 6 小时倒计时通知：活动结束前 6h / 5h / 4h / 3h / 2h / 1h 各发一条推送，始终启用
- 同一时刻多条通知错开 5 秒交错投递，避免系统折叠
- 提醒时间修改后立即刷新通知，无需重启 App
- 活动标记完成 / 静音 / 数据刷新时自动取消并重新注册通知
- 设置页"提醒设置"栏位：开关 + 时间选择器 + 权限状态 + 跳转系统设置 + 一键重新注册 + 测试通知

### 其他
- 入场动画（皇上推门）
- iCloud 同步（SwiftData + CloudKit）
- 后台自动刷新米哈游活动数据

## UI 设计

- 粉色动漫风主题
- 全局波点 Canvas 背景（`Canvas` 绘制 2 pt 圆点， 5% 透明）
- 自定义悬浮底部导航栏（`FloatingTabBar`），选中 Tab 为粉色胶囊；白色底托通过 `.background(Color.white.padding(.top, -40).ignoresSafeArea(edges: .bottom))` 自适应延伸至手机物理底部（home indicator 区域），无需固定高度
- 首页顶部三张总览卡：今日总览（待完成/即将结束/已完成计数）、即将结束（7 天内按结束时间升序，最多 4 条，2 天内红色高亮）、游戏列表（所有关注游戏导航入口）
- 首页采用游戏板块卡片：纯色 Header（含游戏图标、名称、活动数徽章）+ 白色 Body（最多展示 2 条最紧急活动）
- 活动行（`EventRowView`）：白色圆角卡片 + 左侧 4 pt 彩色竖条 + 圆形图标 + 单行标题 + 右侧倒计时胶囊（紧急粉色 / 普通金色）
- 设置页：蓝紫渐变 Header 卡片 + 自定义分区标签 + 圆角白卡布局，替代原生 Form
- 游戏详情页（`EventListView`）：自定义 NavBar（深蓝返回键 + 游戏 emoji + 薄荷"刚刚更新"胶囊）；方形图标 section header；兑换码统一包裹在白色大卡中
- 活动攻略页（`EventGuideView` / `CustomEventGuideView`）：白底波点背景；Toolbar 紧急时显示粉色"⚡ 即将结束"胶囊徽章；英雄卡顶部左侧游戏名深色胶囊 + 右侧 ♥★ 装饰，中部活动标题 + 超大剩余倒计时（44pt），底部日期/分类半透明胶囊；白色 B 站卡（青色 bili 方形 + 薰衣草跳转按钮）；活动详情卡每行带 SF Symbol 图标列（gamecontroller / tag / calendar / clock），粉色 gamecontroller 卡头；"小助手提示"卡（黄色灯泡头 + 3 条黄色数字徽章提示）；底部粉色渐变"标记为已完成"按钮；自定义游戏版本与之前版本视觉完全一致
- 兑换码行（`ExchangeCodeRow`）：彩色状态点 + 等宽大号 code + 描述文字 + 描边"复制"按钮 + 粉色渐变"兑换"按钮

## 技术栈

| 层级 | 技术 |
|------|------|
| UI | SwiftUI，Swift 6 strict concurrency |
| 本地数据 | SwiftData（`UserPreferences`、`CustomGame`、`CustomEvent`） |
| 远程活动 | 社区 API（`hoyo-moe`），JSON 解析 |
| AI 提取 | [Jina Reader](https://r.jina.ai) 抓取网页 → DeepSeek `deepseek-chat` 提取 JSON |
| 攻略视频 | Bilibili Web API |
| 云同步 | CloudKit（自动降级为本地存储） |

## 配置 API Key

在 `Config/Secrets.swift` 中填入：

```swift
enum Secrets {
    static let deepSeekAPIKey = "sk-..."      // 必须：自定义游戏 AI 提取 + 时长查询
}
```

- **DeepSeek**：在 [platform.deepseek.com](https://platform.deepseek.com) 注册获取，中国大陆可直连

`Secrets.swift` 已加入 `.gitignore`，不会提交到 git。

## 使用指南

### 添加米哈游游戏
设置页 → 选择关注的游戏 → 回到首页自动加载。

### 添加自定义游戏
1. 首页右上角 **+** → 填写游戏名和 emoji
2. 在 B 站找到该游戏官方账号发布的**版本综合公告**（含所有活动时间表）
3. 文章页 → 分享 → 复制链接 → 粘贴到链接框
4. 点"AI 提取活动" → 确认结果 → 保存

> 每次游戏更新新版本后，进入游戏详情页 → 点右上角刷新按钮 → 重新粘贴新版本公告链接。

### 活动攻略
点击任意活动行（米哈游或自定义游戏均可）→ 跳转攻略页，可查看相关视频并查询完成所需时长。
